Setup Instructions
1. Install dependencies
pip3 install fastapi uvicorn torch
pip3 install scikit-learn numpy joblib
2. Start server (trains automatically on first run)
cd ~/Desktop/dengue
python3 "dengue pinn.py"
# Trains ~10k epochs then starts serving
# Saves: dengue_pinn13_best.pth
3. Test health endpoint
curl http://localhost:8000/health
# → {"status":"ok","trained":true,...}
